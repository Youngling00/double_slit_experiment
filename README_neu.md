# Doppelspalt-Experiment

Ein interaktives Experiment zur Visualisierung von Welleninterferenz und Quantenmechanik.

```powershell
cd C:\Users\berat\double_slit_experiment

```

## 🔬 Was ist das Doppelspalt-Experiment?

Photonen werden von einer Quelle durch zwei Spalten geschickt. Auf dem Schirm entsteht ein **Interferenzmuster**:
- Wo sich Wellen **verstärken** → helle Streifen
- Wo sie sich **auslöschen** → dunkle Streifen

Die **blauen Punkte** sind einzelne Photonen-Treffer. Mit der Zeit bildet sich das theoretische Muster ab.

## 🎮 Interaktive Kontrollen

- **Spaltbreite:** Ändert die Breite der beiden Spalten (0.05–0.30)
- **Photonen pro Frame:** Wie viele Photonen pro Animation emittiert werden (2–50)
- **Zurücksetzen:** Startet die Simulation neu
